# O'HARVEST (front)

## creation du projet 
---

```
# npm 7+, extra double-dash is needed:
npm create vite@latest . -- --template react

# yarn
yarn create vite . --template react
```

## Activer la config eslint

```
#yarn
yarn init @eslint/config

#npm
npm init @eslint/config
```
